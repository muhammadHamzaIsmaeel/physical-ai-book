## üöÄ Quick Start

### Prerequisites

- **Hardware**: Laptop with RTX 4070 Ti or better GPU, 32GB+ RAM
- **OS**: Ubuntu 22.04 LTS (dual-boot instructions provided for Windows/MacOS users)
- **Skills**: Solid Python programming, basic Linux command line, introductory deep learning
- **No prior ROS or robotics experience required** - everything is taught from zero

### Local Development Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/physical-ai-book.git
   cd physical-ai-book
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start local development server**:
   ```bash
   npm start
   ```
   Opens browser to `http://localhost:3000` with hot reload enabled.

4. **Build for production**:
   ```bash
   npm run build
   ```
   Generates static content into the `build` directory.

5. **Serve production build locally**:
   ```bash
   npm run serve
   ```
