## 3. Validation Checklist (T154)

This checklist helps validate your Premium Lab Tier Unitree G1 robot. Ensure all systems are go before proceeding with advanced research and development.

### Robot Hardware & Basic Control

*   [ ] G1 powers on and performs its initial self-check.
*   [ ] Basic motions (e.g., standing, walking) can be commanded via Unitree's control software or SDK.
*   [ ] All joints respond correctly to commands.
*   [ ] Emergency stop functionality is verified.

### Network & Communication

*   [ ] G1 successfully connects to the dedicated high-speed Wi-Fi network.
*   [ ] You can SSH into the G1's onboard computer.
*   [ ] ROS 2 communication is established between your host PC and the G1.

### Sensor Tests (via ROS 2)

*   [ ] **G1 Built-in Cameras**:
    *   [ ] `ros2 topic list` shows camera image topics.
    *   [ ] `rqt_image_view` or `rviz2` displays live camera feeds.
*   [ ] **G1 Built-in LiDAR/Depth Sensors**:
    *   [ ] `ros2 topic list` shows relevant topics (e.g., `/scan`, `/depth/points`).
    *   [ ] `rviz2` displays live LiDAR/point cloud data.
*   [ ] **Optional External Sensors (e.g., Ouster LiDAR)**:
    *   [ ] ROS 2 drivers are running and publishing data.
    *   [ ] Data viewable in `rviz2`.

### Actuator Tests (via ROS 2 & SDK)

*   [ ] A simple ROS 2 node can send joint position/velocity/torque commands.
*   [ ] Robot responds accurately and smoothly to commands.
*   [ ] Joint state topics (`/joint_states`) are published by the G1.

### Software & ROS 2 Environment

*   [ ] ROS 2 Jazzy is correctly installed and sourced on the G1's onboard computer.
*   [ ] Unitree G1 SDK is installed and configured.
*   [ ] Python 3.10+ is available.
*   [ ] All high-level software (VLA models, task planners) are installed and can load.

### Final Check & Safety

*   [ ] All external sensors securely mounted and connected.
*   [ ] Cables neatly routed and secured.
*   [ ] Workspace is clear and safety barriers are in place.
*   [ ] Battery system is fully functional and charged.

With all checks passed, your Premium Lab Tier Unitree G1 is primed for groundbreaking research in humanoid robotics!
